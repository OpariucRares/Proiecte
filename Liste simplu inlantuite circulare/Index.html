Liste simple înlănţuite circulare

  • O listă înlănţuită este circulară dacă după ultimul element din listă urmează primul element al listei.
  • Nici unul dintre nodurile unei liste circulare nu conţine valoarea NULL în partea de legatură.
  • Pentru a identifica o listă simplu înlănţuită circulară putem reţine adresa oricărui element din listă.
  • Declaraţia structurii care reprezintă un nod din listă va fi:
    • struct nod{
    int inf;
    nod *urm;
};
  • Vom defini tipul de date L, care este adresa unui nod din listă: 
    • typedef struct nod *L;
