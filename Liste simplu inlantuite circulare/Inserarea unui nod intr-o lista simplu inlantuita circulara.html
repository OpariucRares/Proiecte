Inserarea unui nod într-o listă simplu înlănţuită circulară

  • Pentru inserare vom implementa o funcţie denumită Inserare cu trei parametrii:

    1. L – pointer care conţine adresa primului nod din listă în care se face inserarea; acest parametru va fi transmis prin referinţă, deoarece în urma inserării, începutul listei se poate modifica;
    2. p – pointer care conţine adresa nodului din listă după care se face inserarea;
    3. x – informaţia nodului care urmează să fie inserat în listă.

  • La inserare apar două cazuri distincte:

    1. Dacă L == NULL (lista este vidă) noul nod inserat va fi singurul element din listă (deci în partea de legatură vom reţine adresa sa);
    2. Dacă L != NULL inserarea se face în interiorul listei.

Cum se inserează un nod într-o listă simplu înlănţuită circulară?

  1. Vom aloca dinamic memorie pentru un nou nod, a cărui adresă o vom reţine în variabila pointer q. În zona de informaţii vom memora valoarea x (q->inf = x).

  1. Creăm o legatură de la noul nod către nodul care urmează după nodul indicat de p
    (q->urm=p->urm).

  1. Modificăm legătura de la nodul indicat de p; după acest nod va urma noul nod (p->urm = q).

  • Void Inserare(LSI &L, int x, LSI p)

    • {

      LSI q = new nod;

      q->inf = x;

      if (p) { q->urm = p->urm; p->urm = q; }

      else { q->urm = q; L = q; }

    }
